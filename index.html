🎓 GitHub Education Upload Enabler

Version 2.0

A Chrome extension that bypasses the "camera only" requirement on the GitHub Education benefits page, allowing you to upload files (ID cards, proof of enrollment) instead of using a live webcam.

Features

  • 📤File upload bypass — forces the hidden dev_pack_form_camera_required flag to false and injects a custom upload UI
  • 🌐Proxy support — routes GitHub traffic through a configurable proxy via PAC script
  • 🕵️Spoofing — overrides timezone, GPS, language, and WebRTC to appear from a target region
  • 📍Address generator — generates Indonesian addresses for billing form auto-fill

Installation in Chrome

  1. Open Chrome and navigate to chrome://extensions
  2. Toggle Developer mode on (top-right)
  3. Click Load unpacked
  4. Select this project folder
  5. Refresh any GitHub Education page
